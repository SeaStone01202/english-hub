`users` - user profile mapped to `auth.users`

`learning_sessions` - session/thread for each learning flow (`grammar`, `vocabulary`, `mock_test`)

`ai_generations` - log for each Gemini generation call (payload, model, JSON response, token usage)

`session_questions` - generated grammar/mock-test questions in a session

`question_options` - options for `multiple_choice` questions

`user_answers` - user submitted answers per session/question

`vocabulary_items` - generated vocabulary words and examples per session

`vocabulary_progress` - spaced review state for each user-vocabulary pair
